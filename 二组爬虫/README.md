# 爬虫说明：

### 1.环境配置：

> + 使用Scrapy爬虫框架
>   - `pip install scrapy`
> + 使用Scrapyd、SpiderKeeper管理爬虫并发布到服务端
>   + `pip install scrapyd`(安装在服务器端)
>   + `pip install scrapy-client `(客户端)
>   + 配置`scrapy-deploy` 
>   + `pip install spiderkeeper` 
> + 使用Python第三方库jieba 进行中文分词，数据分析
>   + `pip install jieba`
